IDSE103

Game Design and Facilitation

Your group will design an original game and facilitate it. One group will present each class, see syllabus for schedule.

Submit a written set of detailed instructions so that anyone can replicate your game.

Outcomes

  • Energize a group admist a mid-day slump
  • Promote camaraderie and mutal trust
  • Be fun

Constraints

  • No longer than 15 minutes (for 10 players)
  • Scale from 5–15 players
  • Only require materials commonly found in an office (post-its, whiteboard, copier paper, etc)
  • Name your game

Grading

Criteria Points Possible
Facilitation was organized and practiced 25
Players felt energized and bonded 25
Constraints were respected 25
Written instructions are clear and stand on their own 25

Submission

Navigate to your teams folder in the github repository 2018-IDSE103 / Assignments / Game / YOUR TEAM

Submit a pull request to the index.html file that includes your game instructions and any required assets.
